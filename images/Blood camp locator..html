<html>

    <title>Get User Location</title>

  <body>
    <p>Click the button to get your location:</p>

    <script>
      function getLocation() {
        if (navigator.geolocation) {
          navigator.geolocation.getCurrentPosition(showPosition);
        } else {
          alert("Geolocation is not supported by this browser.");
        }
      }

      function showPosition(position) {
        var latitude = position.coords.latitude;
        var longitude = position.coords.longitude;
        document.getElementById("demo").innerHTML =

      }
    </script>
  </body>
</html>

In this example, we have a button that calls the getLocation() function when clicked. The getLocation() function checks if the Geolocation API is supported by the user's browser. If it is, it calls the getCurrentPosition() method to get the user's current position. The showPosition() function is then called with the position data, which is used to display the user's latitude and longitude in the demo paragraph.

Note that the Geolocation API requires the user's permission to access their location, so the user will see a prompt asking for permission when they click the button.
